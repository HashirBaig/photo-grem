# PhotoGrem

A web-based photogrammetry visualization platform designed for exploring and presenting photogrammetric products such as **Digital Surface Models (DSM)** and **Digital Terrain Models (DTM)**.

🌐 **Live Demo:** https://photogrem.netlify.app/

---

## 📌 Overview

**PhotoGrem** is an interactive web application focused on the visualization and presentation of photogrammetry-based geospatial products. The platform is designed to help users explore surface and terrain information through an accessible browser-based interface.

The project presents DSM and DTM outputs generated from aerial imagery and AHN data using tools such as **CloudCompare** and **ERDAS**. It is useful for understanding surface structure, terrain variation, urban change, and related geospatial analysis workflows.

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/HashirBaig/photo-grem.git
```

### 2. Navigate into the project folder

```bash
cd photo-grem
```

### 3. Open the project

Since this is a static website, you can open `index.html` directly in your browser.

```bash
index.html
```

Alternatively, you can run it using a local development server.

Using Python:

```bash
python -m http.server 8000
```

Then open:

```bash
http://localhost:8000
```
